A data visualization dashboard that analyzes GameStop's quarterly and yearly financial reports, transforming complex financial data into interactive charts and diagrams.

## Features

- Interactive financial charts using quarterly/yearly report data
- Clean data visualization with multiple chart types
- Responsive design with modern UI components

## Technical Stack

- Astro for static site generation and performance
- TypeScript for type-safe data processing
- Shadcn/ui for consistent component design
- Recharts for interactive data visualization
